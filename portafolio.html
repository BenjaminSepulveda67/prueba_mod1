Portafolio
Introducción a Html

En este desafío desarrollamos nuestro primer HTML básico, aprendiendo los principios básicos de una estructura bien hecha, incorporación de imágenes, párrafos, enlaces internos, entre otros.

Introducción a CSS

En este segundo desafío desarrollado, nos introducimos en CSS (Cascading Style Sheets) y aprendimos los distintos tipos de estilos CSS: en línea, por ID y por clase.

Introducción a Flebox

En el tercer desafío, aprendimos sobre Flexbox, hablamos del modelo de cajas y cómo alinear elementos de todo tipo mediante Flex.

Introducción a Bootstrap

En este ultimo desafío nos introducimos en Bootstrap, un framework frontend que simplifica el desarrollo web, ofreciendo componentes y estilos listos para usar. Permite crear interfaces modernas y responsivas de manera rápida, asegurando una apariencia profesional en diversos dispositivos.
